isPointInsidePolygon :: Polygon -> Point -> Bool Source #

Tests whether a single 2D point is inside a (concave) polygon

Haskell implementation of the algorithm known as 'even-odd rule' algorithm or 'crossing number' algorithm described in detail at https://wrf.ecse.rpi.edu//Research/Short_Notes/pnpoly.html

## Examples:

>>> let rect = [(0, 0), (1, 0), (1, 1), (0, 1), (0, 0)]
>>> isPointInsidePolygon rect (0.5, 0.5)
True

>>> let rect = [(0, 0), (1, 0), (1, 1), (0, 1), (0, 0)]
>>> isPointInsidePolygon rect (1.5, 0.5)
False
