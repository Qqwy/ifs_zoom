(|>) :: a -> (a -> b) -> b infixl 1 Source #

left-to-right application of a function to a value. ``` (x |> f) = (f x) ```

This is similar to Data.Function.& but more readable. It is based on the same syntax that exists in e.g. F#, Elm, Elixir.
