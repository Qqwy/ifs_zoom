Description

By importing this module you get all functions and types from microlens, as well as the following instances:

  • at for Map and IntMap

  • each and ix for

    • Map and IntMap
    • Array and UArray
    • Seq
    • strict ByteString and lazy ByteString
    • Tree

  • _head, _tail, _init, _last for

    • Seq
    • strict and lazy bytestrings

  • strict and lazy for

    • bytestrings
    • StateT, WriterT, RWST
